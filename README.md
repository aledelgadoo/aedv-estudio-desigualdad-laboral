## 📂 Contenido del Repositorio

Este repositorio contiene todo el ciclo de vida del dato, desde la extracción hasta la visualización final.

* **`ModeloMemoriaProyectoPersonalAEDV...Rmd`**: Código fuente de la memoria narrativa (R Markdown).
* **`DashboardProyectoPersonalAEDV...Rmd`**: Código fuente del cuadro de mandos interactivo (Shiny/Flexdashboard).
* **`index.html`**: Versión compilada de la memoria para su visualización web directa (GitHub Pages).
* **`📁 data/`**: Datasets procesados y limpios provenientes de Eurostat e INE.
* **`📁 scripts/`**: Scripts auxiliares para la carga, limpieza y transformación de datos (ETL).
* **`📁 images/`**: Recursos gráficos y visualizaciones estáticas generadas.

## 🛠️ Tecnologías

El proyecto ha sido desarrollado íntegramente en **R** utilizando:
* **IDE:** RStudio
* **Visualización:** `ggplot2`, `plotly`, `highcharter`, `leaflet`.
* **Despliegue:** `flexdashboard`, `shiny`.
